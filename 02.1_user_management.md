## Gestion des utilisateurs

### !!! Les manipulations dans ce fichier nécésitent les droits admin (root) !!!

### Création d'utilisateur

pour créer un nouvel utilisateur :
```bash 
useradd newusername
```

Je peux créer un utilisateur avec une home directory spécifique avec le -d

Le -m me permet de forcer la création de la home directory

```bash
useradd -d /home/test -m username
```

### Lister les groupes de l'utilisateur

Lister mes groupes
```bash
groups
```

Lister les groupes d'un autre utilisateur
```bash
groups otheruser
```

### Modifier un utilisateur

Ajouter un groupe à un utilisateur
```bash
usermod -aG nouveaugroupe username
```

Supprimer un groupe d'un utilisateur
```bash
usermod -rG anciengroup username
```

### supprimer un utilisateur :
```bash
userdel username
```
supprimer aussi sa home directory :
```bash
userdel -r username
```

### Lister les utilisateurs de la machine : 

```bash
cat /etc/passwd
```

### Créer un groupe

```bash
groupadd nomDuGroup
```

### Supprimer un groupe

```bash
groupdel nomDuGroup
```

### Lister les groupes présents sur la machine

```bash
cat /etc/group
```

### Changer d'utilisateur

```bash
su username -
```

le tiret (optionel) de fin sert à déplacer le terminal 
dans la home directory de l'utilisateur cible.

### L'utilisateur root (admin)

Le root est l'administrateur de la machine, tout ce qui concerne la gestion du système
(ajout d'utilisateur,...) doit être exécuter en tant que root

Pour ce faire on a deux méthodes possibles : 

1) Exécuter une commande en tant que root (Hautement conseillé)

```bash
sudo commande
```

Ici j'exécute la commande en tant que root.

Je peux préciser un utilisteur pour exécuter une commande en tant qu'un autre :

```bash
sudo -u username commande
```

2) Changer l'utilisateur 
Passer en root
```bash
su
```

```bash
su root
```

```bash
sudo su
```
